
 Active Directory Home Lab

   Overview
This project simulates a real-world enterprise IT environment using virtual machines.

  Technologies Used
- VMware Workstation Player
- Windows Server 2022
- Windows 11 Pro
- Active Directory Domain Services

  Key Configurations
- Promoted Windows Server to Domain Controller
- Created domain: corp.local
- Configured DNS for internal communication
- Created and managed users and groups
- Joined Windows 11 client to domain
- Tested authentication with domain users
